# Home Screen Widget

Strimma includes a home screen widget that shows your glucose at a glance without opening the app.

---

## Adding the Widget

1. Long-press on an empty area of your home screen
2. Tap **Widgets**
3. Find **Strimma** in the widget list
4. Drag the widget to your home screen
5. Optionally configure the widget appearance

---

## What the Widget Shows

- **Glucose value** — large, color-coded text (cyan/amber/red/gray)
- **Direction arrow** — trend direction
- **Delta** — how much glucose changed
- **Mini graph** — a compact glucose history graph

Tap the widget to open the Strimma app.

---

## Widget Configuration

When you add the widget (or tap the configuration option), you can adjust:

- **Background opacity** — how transparent the widget background is, from fully transparent to fully opaque

---

## Widget Updates

The widget updates automatically whenever Strimma receives a new glucose reading. Updates are near-instant — the widget reflects the same data as the main app and notification.

If the widget shows stale data, check that:

- Strimma's foreground service is running (notification visible)
- Battery optimization is disabled for Strimma
- Your CGM app is posting notifications
